# Starbucks-webpage
Starbucks Website Clone
A professional, responsive Starbucks-inspired website developed using HTML, CSS, and JavaScript. This project demonstrates modern frontend design principles with clean UI, smooth layout, and structured sections such as Home, Shop, Delivery, App, About, and Contact.

Overview
This project recreates the look and feel of a Starbucks-style landing website to practice frontend development skills. It focuses on layout design, responsive structure, and user-friendly navigation while maintaining a professional appearance suitable for portfolio use.

Features
*Responsive and clean UI
*Modern landing page design
*Product showcase section
*Delivery service section
*App promotion section
*About and contact sections
*Smooth navigation experience

Technologies
*HTML5
*CSS3
*JavaScript
